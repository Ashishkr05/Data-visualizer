This platform is about to analyze the data of the customer/User and the data of the products that has been stored in sql server database. The New User will be able to see all the customer's username previously visited in that shop.


The developer use data-file of products and upload it on website. The role of developer is to provide the data to the user. Developer uploads the data in different shops on the website. User can select any shop given on the website. The platform provides user more precise information about the products to have ease in products selection. The user can select multiple items from the shop.


The User can see the relevent items of the selected items as recommendation of the selected items. The user will get idea about which products to select. At last If User wants to purchase the selected items then he/she can use buy button. After that User will be back to the homepage. The selected items will be stored in the database.


For the record of user data the platform has User page there. The User page will appear when customer is done with selection of shop. In the UserPage customer has to create a Username. The record of every customer will be stored in the database.
